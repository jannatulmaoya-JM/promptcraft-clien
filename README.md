# PromptCraft Dashboard

This is a clean, role-based dashboard application built for managing, creating, and reviewing prompt templates efficiently.

##  Project Access
 **Live Side:** [https://promptcraft-clien-git-main-jannatulmaoya-jms-projects.vercel.app/]
 **Client Side:** [https://github.com/jannatulmaoya-JM/promptcraft-clien]
 **Server Side:** [https://github.com/jannatulmaoya-JM/promptcraft-server]

##  Admin Access (For Testing)
 **Email:** `admin@example.com`
 **Password:** `your-secure-password`

##  Role-Based Dashboard Access
Depending on the user role, the sidebar will show different tools:

 **User:** Access to Profile, My Prompts, Saved Prompts, Add Prompt, and Reviews.
 **Creator:** Access to Dashboard Home, Create/Add Prompts, and My Prompts management.
 **Admin:** Full control over All Users, All Prompts, Payment Logs, Reports, and Analytics.

##  Tech Stack
 **Framework:** Next.js (App Router)
 **Design:** Tailwind CSS
 **Components:** HeroUI (NextUI)

##  How to Run Locally
1. Clone the repo and get into the folder.
2. Run `npm install` to set everything up.
3. Use `npm run dev` to start the local server.
